# baseProject
基于ant-design-vue@next和Vue3.0，构建一整套开发框架，包含按需引入，svg封装，路由无限极嵌套及管理，登录管理，权限管理等，样式暂时不定制，尽可能使用ant-design-vue的样式表文件。

# node@14.15.1
# npm@6.14.9
# vue@vue/vli 4.5.9
# ant-design-vue@2.0.1
